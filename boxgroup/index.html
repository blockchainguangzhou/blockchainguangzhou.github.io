        # 为什么投资区块链

        区块链技术是目前和未来的朝阳行业之一，这在习大大最近的讲话上也得到了印证。错过今日的区块链相当与十九世纪末错过互联网。

        • 亚马逊的股票(AMZN)从1997年5月16日的1.73美元（前复权），涨到了2019年11月29日的1800美元。

        • 苹果的股票(AAPL) 从1996年8月12日的0.74美元上涨到2019年11月29日的267美元。

        • Facebook的股票(FB)的股票从2012年5月18日的38.23美元上涨到2019年11月29日的201美元。

        从今天往回看，互联网行业公司在经历10-20年的发展之后，相当多公司股价有着几十倍到几千倍不等的涨幅。今日的区块链就像当年的互联网，若不参与，则会错失未来潜在的巨大收益。

        # BOX为什么要选择Bitcoin, EOS和XIN作为标的

        目前区块链的发展还出于初级阶段，整个行业鱼龙混杂，作为普通人根本无法分辨出真正具有投资价值的。

        不同于其他领域，在投资领域盲目试错危险，在区块链世界试错尤为危险。不信你去 Coincapmarket.com 上看看，截止2019年12月初，总共2348个币种，多少币的市值能从2017年最高峰跌去95%。若是在当初满仓，现在损失不可想象。

        你可能会说：比特币当初也跌很多啊，现在价格不也挺高的吗，只要我不卖，那这账面损失就不会实际兑现啊。 但事实上，这2348种币，绝大多数币种对应的区块链应用根本毫无价值，有些币种的白皮书干脆就是花了千元请人代写，这意味着买了这些币的人的账面损失将变成事实损失。

        如此行业乱象，亟需一个掌舵人避开重重陷阱，分析出未来发展趋势。

        这就是李笑来。李笑来通过总结近10年来投资经验，认为区块链的发展趋势为：

        可信账簿 --> 可信代码 --> 可信执行环境 --> 可信硬件。

        • Bitcoin：第一个通过使用区块链技术完成“公开透明不可篡改的账簿“ — — 可信账簿；市场上最被看好的标的

        • EOS：多线程版本的“公开透明不可篡改的代码平台“；Block.one 是当下区块链世界最具技术实力和最具资金实力的公司。

        • XIN：第一个通过使用类区块链技术完成“公开透明不可篡改的可信执行环境”；第一个提供大幅度降低用户进入区块链门槛的去中心化服务 ---- 6 位数字密码即私钥，兼具安全和易用性让区块链以更加友好方式的面向亿万用户。

        从这里能够看出，BOX则代表了这个行业的发展趋势，而且稀释了机会成本，且无任何手续费、没有投资抽水，对于普通人是投资区块链的最佳选择，没有之一。

        # 我自己可否按照比例购买？

        理论上可以，实际上很难。

        1. 操作上更加麻烦，BOX只要买一种，分开来要买三种。

        2. 单单是“麻烦”这一条，能使得90%以上的人无法做到长期投资。

        # 为什么定投是最好的方式？

        最关键的原因是：币市熊市很长，牛市很短。只有通过定投才能够在相当长的时间内在熊市中在低价购买足够多的砝码。也就是说，定投对于普通人而言是简单、粗暴、有效、且正确的投资方式。定投让普通人长期持有优质标的，不因诱惑而进行频繁的买入卖出操作。

        # 我要定投多久？每次定投多少？

        强烈推荐定投两个大周期（区块链世界中的两个大周期为6-8年），定投的资金数量与频率完全由自己把控。通常建议把未来3年内不用的闲钱用于投资，每周或每月定投均可。

        # 进了践行群以来，作者到底发生改变了吗？

        之前的我和大多数人一样，追涨杀跌，一地鸡毛。我也买过空气币，损失过五位数的人民币。我盼望着牛市的到来，希望卖在最高点，每当看到价格上涨，都异常兴奋。

        进群会后，有了长期的概念，我倒是希望熊市能够越长越好，因为只有熊市足够长，我才能有足够长的时间购买足够多的低价筹码，在日后的牛市中，有足够多的盈利。

        现在的我，每当看到价格下跌，会异常的兴奋。

        除了，区块链投资外，更重要的是我的人生态度发生了变化，我思考问题具备了长期的视角，因此做的决策也和之前完全相反：

        我相信很多人和之前的我一样，对于事业编制有一种迷恋，认为稳定的就是好的，不用担心失业，有国家为你养老。这听起来很美好对不对？

        但是我实际工作之后才知道，这里面要处理很多险恶的人际关系，开不完的无聊会议，工资收入也不算高。在这里得到了稳定，但却会失去比稳定重要的多的东西：成长。

        成长，正是从长期看来最重要的东西。

        经过了这么多课程的洗礼，我决定了辞职事业单位，并换了一份更有成长性的工作。同时还开展了我的副业——创建了区块链广州组织。

        我从零开始，学Python，Markdown，VuePress，Github，从零开始建立个这个如你所见的网站。它现在还不完美，但这没关系，我有长期的视角，我会让它慢慢变好，慢慢成长。
